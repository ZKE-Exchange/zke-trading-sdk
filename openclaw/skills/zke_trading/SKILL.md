# ZKE Trading Skill

This skill enables AI agents to interact with the ZKE Exchange trading toolkit.

Supported capabilities include:

- Spot market data
- Spot trading
- Futures market data
- Futures trading
- Margin trading
- Withdraw requests
- Withdraw history
- WebSocket market streams

This skill works together with the **local MCP server** included in the ZKE Trading SDK.

## Requirements

Before using this skill:

1. Install the ZKE Trading SDK locally
2. Configure your API keys in `config.json`
3. Start the MCP server
